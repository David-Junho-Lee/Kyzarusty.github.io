Describe what the DOM is and an example of how you might interact with it

The Document Object Model(DOM) is a programming interface for web documments. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page



We can access these elements in the document and make changes to them using JavaScript.

Explain the difference between accessing data from arrays and objects

Arrays use zero-based indexing(first item in an array has a index of 0) and can be accessed by their numerical position in the list using bracket notation

Whereas objects use properties and can be accessed by their key in the list using either dot notation or bracket notation.

Explain what functions are and why they are helpful

Function is a block of reusable code to perform any particular task. Functions are useful as we can reuse them instead of writing same routine again and again. Also it helps to reduce debugging time as there will be less block of codes to deal with, end up saving a lot of time.
